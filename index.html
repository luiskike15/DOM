DOM(OBJETO DOCUMENTO)

Que es el DOM

Es una representación estructurada del documento que permite a los programas y scripts acceder y manipular dinámicamente el contenido, la estructura y el estilo de un documento. Es esencial para la interactividad de las paginas webs.

Por ejemplo si tenemos un texto 'Hola' parrafo y al dar clic a un boton cambiamos a otro texto que diga "Que tal". Se puede realizar ello gracias al DOM.

Obtener Elementos

Que son los nodos

Todo elemento html esta conformado por nodos, los mas importante son los nodos de texto, nodo de elemento y nodo de atributo.

  • Nodos de texto: Es todo el contenido que va dentro de una etiqueta p por ejemplo.
  • Nodos de elemento: Refiere a las etiquetas html
  • Nodos de atributo: Refiere por ejemplo a las clases, id.
