#Que es el DOM
Es una representación estructurada del documento que permite
a los programas y scripts acceder y manipular
dinámicamente el contenido, la estructura y el
estilo de un documento. Es esencial para la interactividad
de las paginas webs.

#Obtener Elementos
Para obtener elemento y poder modificarlo puede ser
mediante el id que se utilizaria (getElementById), 
si quiero agarrar un selector, es decir agarrar 
el primer elemento que coincide con un 
selector utilizaria (querySelector) o  si quieres agarrar 
varios elementos (querySelectorAll).

#Que son los nodos
Todo elemento html esta conformado por nodos, los mas
importante son los nodos de texto, nodo de elemento y nodo
de atributo.

#text Content
Es similar al innetHTML mas no igual, no solo trae todo el texto
de la etiqueta sino mas bien muestra todo lo que tiene , ademas el script.

#innerHTML
Es una propiedad que permite ingresar y a la vez modificar el valor que hay dentro del html y poner un nuevo contenido, solo muestra el texto.

#Estilos
Mediante Javascript podemos modificar los estilos de las etiquetas html,
ya sea color fondo, color fuente, tamano, padding, etc.

#DOM Traversing
Sirve para moverse dentro del DOM, para acceder, buscar un dato o elemento. 

#CreateElement
Es un metodo para crear un elemento html  bajo el DOM y se crea con createElement.
